the canonical `AGENTS.md` testing policy prohibits automated agents from
running a workspace-wide Clippy gate unless the maintainer requests that exact
gate. A targeted protocol-surface Clippy run passed, but it is not substituted
for the frozen command. This post-freeze conflict is
`CANIC-092-AUDIT-011`.
