# Release-Line Audits

This directory holds numbered-line closeout audits and cross-line program state
audits.

Use this directory for reports that summarize or close a release line, such as:

```text
0.51-closeout.md
41-50-program-state.md
```

Do not put recurring dated audit runs here. Those belong under:

```text
docs/audits/reports/
```

Do not put reusable audit templates here. Those belong under:

```text
docs/audits/recurring/
docs/audits/modular/
```
