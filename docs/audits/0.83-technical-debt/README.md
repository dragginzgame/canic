# Canic 0.83 Technical Debt Audit

This directory contains the docs-only audit artifacts for the 0.83 technical
debt line.

- `ledger.md` is the canonical human-readable debt ledger.
- `recommended-slices.md` groups accepted findings into future focused slices.
- `deferred.md` tracks deferred findings and rationale.
- `rejected.md` records inspected candidates that are not debt.

0.83 is audit-first. Findings do not authorize fixes by themselves; fixes need
a separate accepted implementation slice.
