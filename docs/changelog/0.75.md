# 0.75 Changelog

Note: `0.75.0` was accidentally published and yanked. It is intentionally
skipped in this changelog; `0.75.1` is the first supported `0.75` release.

## 0.75.1 - 2026-06-28 - Release publishing recovery

### 🩹 Fixed

- Release publishing now checks the exact `crate@version` on crates.io before
  skipping or waiting on a workspace crate.

- This fixes recovery publishing when a newer or yanked version exists, because
  `cargo search` only reports the displayed crate version rather than every
  published version.
