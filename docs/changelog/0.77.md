# 0.77 Changelog

## 0.77.0 - 2026-07-01 - Wasm footprint feature boundaries

### 📝 Summary

- Starts the 0.77 line with a targeted wasm-footprint cleanup pass after the
  delegated-auth hard cut.

- Keeps delegated-token verification, root renewal, issuer behavior, root
  publication, endpoint surfaces, stable records, and wire contracts unchanged.

- Reduces shared wasm fan-in by making heavyweight auth and control-plane
  dependencies explicit feature choices instead of unconditional build input for
  store/control-plane-only wasm artifacts.

### 🔧 Changed

- Chain-key ECDSA support is now feature-gated so delegated-token verifier and
  root-renewal builds keep secp256k1 verification/signing, while
  control-plane/store-only wasm builds avoid linking `k256`.

- The `canic` facade exposes separate chain-key feature switches for ECDSA
  verification and root signing so root canisters, delegated-token verifiers,
  native host validation, and store-only canisters can select the smallest
  compatible feature set.

- The control-plane crate now separates root bootstrap/publication orchestration
  from the local wasm-store canister API.

- `canic-wasm-store` now depends on the local wasm-store canister feature
  instead of the full root control-plane surface.

- Direct test/helper crates that depend on `canic-control-plane` explicitly opt
  into the full root-control-plane plus wasm-store feature set, preserving their
  previous API surface after workspace dependency defaults were tightened.

### 📊 Audit

- The first same-day wasm footprint rerun showed the chain-key ECDSA split
  reducing the packaged wasm-store gzip artifact by about 17 KB and the root
  bundle's shrunk wasm by about 19 KB versus the original 2026-07-01 baseline.

- The follow-up same-day wasm footprint rerun after the control-plane split
  kept leaf canister sizes flat while reducing the root bundle's shrunk wasm by
  about 32 KB versus the original 2026-07-01 baseline.

- The latest packaged wasm-store gzip artifact measured about 844 KB.

- Detailed footprint report:
  `docs/audits/reports/2026-07/2026-07-01/wasm-footprint-3.md`.
