### Verified

- The smoke runs in a temporary project and covers:

  ```text

  canic scaffold canister demo store
  canic fleet role inspect demo store
  canic fleet role attach demo store --subnet prime
  canic deploy catalog list --format json --output catalog.json
  canic evidence gate --policy policy.toml --envelope envelope.json --format json --output gate.json
  ```

## 0.55.0 - 2026-05-31 - v1 stabilization design

### Added

- Added the 0.55 v1 stabilization design:

  ```text
  docs/design/0.55-v1-stabilization-readiness/0.55-design.md
  ```

### Clarified

- Defined 0.55 as a stabilization/readiness line rather than a new feature
  expansion. The line exists to prove the compact v1 operator story is
  understandable, testable, and release-ready before adding deployment groups,
  signing, locks, registry import, teardown, controller mutation, active
  adoption/import, or broad live verification.

- Named the compact v1 command set that 0.55 should prove:

  ```text
  canic fleet create <fleet>
  canic scaffold canister <fleet> <role>
  canic fleet role attach <fleet> <role> --subnet <subnet>
  canic build <fleet> <role> --provenance <path>
  canic deploy check <deployment> --format envelope-json
  canic evidence gate --policy <path> --manifest <path>
  canic deploy catalog list
  canic deploy catalog inspect <deployment>
  ```
