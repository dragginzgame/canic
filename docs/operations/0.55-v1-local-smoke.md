# 0.55 V1 Local Smoke

This smoke check proves the local, no-network subset of the compact v1
operator story. It is intentionally small and runs in a temporary project.

Run it from the repository root after building the local CLI:

```bash
cargo build -p canic-cli
scripts/ci/v1-readiness-smoke.sh
```

To use a different `canic` binary:

```bash
CANIC_BIN=/path/to/canic scripts/ci/v1-readiness-smoke.sh
```

## What It Covers

The script creates a temporary workspace and runs:

```text
canic fleet create demo --yes
canic scaffold canister demo store
canic fleet role inspect demo store
canic fleet role attach demo store --subnet prime
canic fleet role inspect demo store
canic deploy inspect catalog list --json --output catalog.json
canic evidence gate --policy policy.toml --envelope envelope.json --json --output gate.json
```

It asserts:

- fleet creation succeeds;
- scaffolded roles start declared-only;
- declared-only roles are blocked from deploy artifact builds;
- explicit attachment makes the role build-eligible;
- catalog output stays local-state-only and reports an empty deployment list
  when no deployment-target state exists;
- evidence gate evaluates one saved `EvidenceEnvelopeV1` without running
  builds, deployments, live discovery, or mutation.

## What It Deliberately Does Not Cover

The smoke does not run:

```text
canic build <fleet> <role> --provenance <path>
canic deploy check <deployment> --evidence-envelope
canic install <app> <fleet>
```

Those paths require a real Cargo/Wasm build or an installed deployment target.
They are covered by focused unit tests and existing local operator flows, but
this smoke avoids them so it can stay fast, local, and mutation-safe.

For the heavier build-provenance and deployment-check rehearsal, use:

```text
docs/operations/0.55-v1-operator-proof.md
scripts/ci/v1-operator-proof.sh
```

The smoke also does not add:

- deployment groups;
- signing;
- locks;
- registry import;
- teardown;
- controller mutation;
- active adoption/import;
- broad live verification.

## Expected Result

Successful output ends with:

```text
v1 readiness smoke passed
```
