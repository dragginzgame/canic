## Delegated Session State

- [x] Store one active delegated session per wallet per canister.
- [x] Persist session records across upgrade until expiry.
- [x] Support explicit session clear/logout.
- [x] Ignore expired sessions automatically.
- [x] Fall back to raw caller when no active session exists.
- [x] Reject obvious canister/infra principals as delegated session subjects.
