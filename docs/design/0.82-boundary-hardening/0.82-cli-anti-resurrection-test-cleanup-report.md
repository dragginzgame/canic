## Findings

### CANIC-082-CLI-ANTI-RESURRECTION-TEST-CLEANUP-001

Severity: P3
Category: test_ownership
Owner: `canic-cli`
Current location: CLI unit tests
Intended location: current-surface behavior tests only

Evidence:
Inspect, medic, state, deploy-plan, and top-level CLI tests included blocks
that asserted removed aliases or historical rejected forms remain rejected.

Risk:
Anti-resurrection tests keep obsolete command names and option shapes visible
inside the maintained test surface, creating review noise and encouraging
future changes to preserve old vocabulary.

Decision:
Fixed with a no-behavior-change cleanup. Removed obsolete-command tests while
keeping current positive parser, help, report, JSON, and exit-code coverage.

Regression test:
Not applicable. This slice removes tests for removed surfaces and does not
change command behavior.
